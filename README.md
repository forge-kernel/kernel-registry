# Forge Kernel Registry

Version archive for the Forge kernel. The standalone `install.php` script
(located in the project root) uses this registry to install, upgrade, or
downgrade the kernel.

```bash
php install.php latest
php install.php 0.1.1
```

If the kernel is ever in a bad state, you can always fix it here without
needing the package manager.

This is a personal project. No pressure, no bloat, no hype. Built for
builders who want full control. Fork it, remix it, break it apart.
